## Task2 표시 교정(가역)
- DRUG_COLS: `{key:'edi_price', label:'단가', default:true}` + `{key:'price_unit', label:'통당단가', default:false}`(보조 열). 'EDI단가'(insurance_price) 그대로(default:false).
- cellVal: `edi_price` 분기 신설(구입단가, 빈값 '-'). `price_unit` 분기는 '통당단가' 열로 유지.

## Task3 인라인 편집(가역·안전)
- DrugList 내부: `canEditPrice = admin/owner` (useTheme의 profile·memberRole). 셀 클릭→number input→Enter/blur 저장.
- `saveEdi`: 입력검증(공란·NaN·음수 차단) 후 `supabase.from('drugs').update({ edi_price })`(**edi_price 단일 컬럼만**, RLS 테넌트 격리). 저장 성공 시 로컬 override로 즉시 반영.
- member(읽기 전용)는 클릭 편집 불가(텍스트만). 서버 RLS 역할 강제는 후속 제안(현재 UI 게이트).
- 이력(누가·언제·얼마→얼마): 이번 미구현, 후속 별도 제안.
