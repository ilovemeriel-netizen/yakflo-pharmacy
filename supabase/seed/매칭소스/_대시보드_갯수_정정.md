## 수정 (가역·클라이언트만)
- App.jsx:701 본문 '갯수' 값 `c.qty`(SUM) → **`c.total`(COUNT, 헤더와 동일 변수)**.
- 기존 수량합은 제거하지 않고 **'총 재고수량'** 라벨로 분리 표시(c.qty). (사용자 Q1 확정)
- catData(666)는 total·qty 둘 다 이미 계산 → 데이터/RPC/상태필터 무변경. 헤더·갯수 동일 소스(active=사용)라 분모 일치.
- 되돌림: 본 커밋 revert(1줄 치환 복원). RPC·DB 변경 없음 → SQL 롤백 불요.
